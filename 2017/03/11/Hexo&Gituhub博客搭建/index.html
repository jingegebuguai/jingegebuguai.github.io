Hexo与Github连接

deploy部署

deploy:
      type: git
      repo: git@github.com:###/###.github.io.git
     branch: master

其中###为github账户名，流程开始：

  • 在git博客到gihub前在git bush中写一行如下代码，避免出错

如下所示：

$ npm install hexo-deployer-git --save
  • 再次进行git部署

这次是保险起见，因为在根目录配置文件下部署经常失效，如下所示：

$ cd d:/hexo/blog
$ git clone https://github.com/###/###.github.io.git .deploy/###.github.io
  • 博客导入github详细过程

由于我们每次新建（$hexo new “file”）所产生的 .md 文件都在\hexo\blog\source\_posts目录下面,之后可以利用markdown进行编辑。所以我们创建一个deploy脚本文件，如下

hexo generate
cp -R public/* .deploy/###.github.io
cd .deploy/###.github.io
git add .
git commit -m “update”
git push origin master

hexo generate将我们之前的 .md 生成public文件夹下的新内容，然后将其拷贝(cp -R)至###.github.io的git目录下，然后使用git commit命令提交代码到###.github.io这个repo的master branch上。这个时候再去github就能看到写的博客，而利用域名的话，如果已经拥有域名，只要将域名解析再添加到###.github.io/setting/Custom domain/即可。

关于Hexo—admin

这是一款与Markdown性质一样的插件，但是比Markdown更加兼容Hexo通过这款插件，可以很方便地管理所有的文章。可能会有人问，这样子和简书之类的博客不就相同了吗，管理上更像了，但是也更方便了。
写在前面，如果使用时linux系统，这款插件很完美，如果使用的windows系统，建议不要使用这款插件，会出各种各样的问题。

cd yourblog
npm install --save hexo-admin
hexo server -d

这时打开http://localhost:4000/admin/就能看到相应的界面，提交也只要输入'deploy',就可以直接上传到Github。

有关Next主题的美化

###canvas-nest.js特效
就像本博客背景那些粒子一般，设置很简单，但是新手也难免绕很大一个弯。先去博客clone canvas-nest.js包，然后将canvas.js置于\themes\next\source\js\src下，再在\themes\next\layout\_lanyot.swig文件下添加如下代码

<script type="text/javascript" color="255,127,80" opacity='0.7' zIndex="-2" count="150" src="//cdn.bootcss.com/canvas-nest.js/1.0.0/canvas-nest.min.js">
</script>

其中颜色尺寸数量根据自己喜好配置就行，但是这很占内存。最后执行下面命令效果就会出现。

hexo clean
hexo -g
hexo -s

###Next主题色彩等配置
打开hexo/themes/next/source/css/_variables/base.styl，代码如下：

$whitesmoke    = #f5f5f5
 $gainsboro     = #eee           //这个是边栏头像外框颜色，
  $gray-lighter  = #ddd           //文章中插入图片边框颜色
  $grey-light    = #ccc           //文章分割线、下划线颜色
  $grey          = #bbb           //页面选中圆点颜色
$grey-dark     = #999
$grey-dim      = #FF6666     //侧边栏目录字体颜色
$black-light   = #6699CC     //修改文章字体颜色
$black-dim     = #333
$black-deep    = #6699CC     //修改主题的颜色
$red           = #ff2a2a
$blue-bright   = #87daff
$blue          = #0684bd
$blue-deep     = #262a30
$orange        = #fc6423     //浏览文章，目录选中的颜色

背景图片色彩

打开hexo/themes/next/source/css/_custom/custom.stl，没有的话新建一个，添加代码，如下：
背景颜色直接更改即可：body { background: #F0F0F0; }
添加背景：body { background: url（’/images/background.jpg’); }(将背景图片放到hexo/source/images中即可。

// Custom styles.
body{
    background-color:#F5F6EB 
     #background: url（'/images/background.jpg')
}

二者最好取一，因为可能会有覆盖。
